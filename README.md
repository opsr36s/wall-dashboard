# Wall Dashboard 🖥️

Настенный информер для старого iPad. Один файл `index.html`, чистый HTML/CSS/Vanilla JS, без сборки и без фреймворков. Все API — бесплатные, без ключей, с открытым CORS. Крутится 24/7 в Safari.

## Особенности

- **5 цветов**, тёмная спокойная палитра, плиточный «гаджетный» дизайн.
- Кэш в `localStorage` с TTL — API не дёргаются лишний раз, при сбое показывается последнее значение.
- Автообновление (настраивается, по умолчанию 10 мин); МКС — каждые 15 с; лента новостей ротация 12 с.
- Wake Lock — экран не гаснет; двойной тап — полноэкранный режим.
- Скрытая шестерёнка ⚙ внизу справа — смена города/координат/интервала.
- Адаптив: горизонтальный (1024×768) и портретный режимы.

## Деплой

**Netlify (без регистрации, 30 сек):** открой <https://app.netlify.com/drop> и перетащи эту папку. Получишь ссылку `имя.netlify.app`.

**GitHub Pages:**
```bash
git remote add origin https://github.com/USERNAME/wall-dashboard.git
git push -u origin main
# затем: Settings → Pages → Branch: main / root → Save
```
Адрес: `https://USERNAME.github.io/wall-dashboard/`

## На iPad

Открой адрес в Safari → «Поделиться» → «На экран Домой». Запускай с домашнего экрана — будет полноэкранно, без панелей браузера.
